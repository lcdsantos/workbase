WorkBase

A solid foundation with sensible defaults to kickstart your next project. Start building from the ground up. Not the other way around.

  • Gulp for task running
  • Sass, with various mixins for style.
  • webpack with Babel for JavaScript
  • EJS for HTML templating
  • Browsersync for live-reloading

Getting started

Clone the repository

$ git clone https://github.com/lcdsantos/workbase.git my-project && cd my-project

Install gulp globally

$ npm install gulp-cli --global

Install dependencies

$ npm install

Start working! Run gulp and open http://localhost:3000 on your browser

$ gulp
